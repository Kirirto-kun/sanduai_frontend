ТЕХНИЧЕСКОЕ ЗАДАНИЕ (ТЗ) №1  
на разработку программного обеспечения «Платформа для учителей» (Sandu AI)  
**1\. ОБЩИЕ СВЕДЕНИЯ**

1.1. Наименование Проекта: Образовательная платформа с ИИ для учителей (рабочее название «Sandu AI»).  
1.2. Стороны:

* **Заказчик:** ИП «СЕЙТКАЛИЕВ МУРАТ ШАМГАНОВИЧ»  
* Исполнитель: ИП «Мажитов Джафар»  
  1.3. Цель разработки: Создание автоматизированного сервиса для помощи педагогам в генерации учебной документации, загрузке и просмотре образовательных материалов, а также интеграция платежных систем и медиа-генерации.

**2\. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ**

Система представляет собой веб\-приложение с интуитивно понятным интерфейсом (кнопки, меню), включающее следующие основные модули:

### **2.1. Модуль «ИИ-Функции» (Текстовая генерация)**

Платформа должна обеспечивать генерацию контента по запросу пользователя с возможностью экспорта в форматы **Word (.docx)**, **LaTeX** и **PDF**.

**Список функций генерации:**

1. **КМЖ (Күнтізбелік-тақырыптық жоспар):** Адаптировано под концепцию «Құндылыққа негізделген адал азамат».  
2. **ЭССЕ:** Генерация сочинений на заданную тему.  
3. **МАҚАЛА:** Написание статей педагогической или предметной тематики.  
4. **БЖБ (Бөлім бойынша жиынтық бағалау):** Составление заданий СОР.  
5. **ТЖБ (Тоқсандық жиынтық бағалау):** Составление заданий СОЧ.  
6. **Ғылыми жобалар:** Структурирование и написание основ научных проектов.  
7. **Классные часы:** Сценарии воспитательных уроков.  
8. **Рабочие листы:** Генерация раздаточного материала с заданиями.  
9. **Балабақшаға ашық оқу іс-әрекеттері:** Сценарии занятий для дошкольных учреждений.  
10. ИИ тесты  
11. Интерактивные игры в виде Кахут  
12. Озвучка при помощи ИИ

**Функционал интерфейса:**

* Ввод темы запроса текстом или выбор из меню.  
* Кнопка «Получить в Word/PDF» после генерации текста.

### **2.2. Модуль «Библиотека материалов» (Контент заказчика)**

Платформа для размещения готовых материалов, загружаемых Заказчиком (Администратором), с доступом для пользователей (Подписчиков).

**Типы материалов:**

1. **Курстар (Курсы):** Видео-уроки или текстовые модули.  
2. **Көрнекіліктер (Наглядные пособия):** Плакаты, схемы, раздаточный материал.  
3. **Интерактивті презентациялар:** Слайды, презентации.  
4. **Интерактивті ойындар:** Обучающие игры.  
5. Скетч-Хаб  
6. Генерация презентаций при помощи ИИ

**Функционал:**

* **Админ-панель:** Для загрузки материалов Заказчиком.  
* **Пользовательский интерфейс:** Просмотр и скачивание материалов.  
* **Фильтрация и Поиск:** Удобный поиск по предметам, классам, темам.

### **2.3. Модуль «Медиа и Платежи»**

1. **Генерация Фото:** Создание изображений по запросу через ИИ.  
2. **Генерация Видео:** Создание коротких видеороликов.  
3. **Цифровой Аватар:** Функция оживления персонажа или диктора.  
4. **Платежная система:**  
   * Оплата банковской картой.  
   * Интеграция Kaspi QR.

**3\. ЭТАПЫ РАЗРАБОТКИ И СРОКИ**

Разработка осуществляется в 3 (три) этапа согласно следующему плану:

### **ЭТАП 1: Разработка текстового ИИ-ядра**

**Задачи:**

* Реализация структуры бота (Меню, Кнопки).  
* Разработка и настройка промптов (инструкций) для всех текстовых функций:  
  * Функции КМЖ, Эссе, Макала, БЖБ, ТЖБ, Ғылыми жобалар, Классные часы, Рабочие листы, Детский сад.  
* Реализация конвертера форматов: Текст \-\> Word (.docx), Текст \-\> LaTeX/PDF.  
* Функционал скачивания готовых документов.

**Результат этапа:** Рабочая платформа, генерирующий документы по запросу.

### **ЭТАП 2: Загрузка материалов и База знаний**

**Задачи:**

* Разработка бэкенда для хранения файлов.  
* Создание интерфейса загрузки (для Заказчика) категорий: Курсы, Көрнекіліктер, Презентации.  
* Реализация просмотра материалов для пользователей.  
* Внедрение системы фильтров и поиска по базе материалов.

**Результат этапа:** Наполняемая библиотека контента с доступом для пользователей.

### **ЭТАП 3: Финтех и Мультимедиа ИИ**

**Задачи:**

* Подключение платежного шлюза (Эквайринг карт).  
* Интеграция API для генерации Kaspi QR (при наличии технической возможности со стороны банка) или ссылок на оплату.  
* Интеграция нейросетей для генерации изображений и видео.  
* Реализация функции «Цифровой аватар».

**Результат этапа:** Полностью готовый продукт с монетизацией и медиа-функциями.

**4\. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ**

4.1. Платформа: Интерактивный Сайт. 4.2. Язык интерфейса: Казахский / Русский (выбор языка).  
4.3. Доступность: 24/7, за исключением периодов технических работ.  
4.4. Безопасность: Защита данных пользователей, безопасное соединение при оплате.  
**5\. ПОРЯДОК ПРИЕМКИ**

5.1. Приемка осуществляется по завершении каждого этапа.  
5.2. Заказчик проверяет соответствие функционала пунктам 2 и 3 данного ТЗ.
